# 课程路线图：7 年前端架构师从零学 Flutter

## 课程定位

这不是“API 清单”，而是用前端架构师熟悉的概念迁移到 Flutter：

- React/Vue 组件树 → Flutter Widget Tree
- CSS 布局与盒模型 → Flutter 约束布局模型
- hooks / component state → `StatefulWidget`、`setState`、`ValueNotifier`
- SPA Router → `Navigator` 与 Route
- API Client / Repository → Flutter 中的异步 UI 与数据层边界
- Testing Library / component test → `flutter_test` widget test

## 学习节奏

每课建议 60-90 分钟：

1. 读课程文档，建立心智模型。
2. 跑 demo，观察 UI 与状态变化。
3. 修改一个小需求，训练迁移能力。
4. 执行 `flutter test`，确认行为没有破坏。
